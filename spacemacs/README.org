* Overview

This directory contains the Spacemacs configuration used in this project.

* Setup

** Automated Setup (Recommended)

Use the ~./ensure.sh~ script under ~devtools/setup-dev/ansible/~ directory:

#+BEGIN_SRC bash
cd devtools/setup-dev/ansible
./ensure.sh spacemacs
#+END_SRC

This will set up Spacemacs with all necessary configurations and dependencies.

** Available Playbooks

- ~spacemacs.yml~ - Main Spacemacs setup
- ~spacemacs-go.yml~ - Go development configuration for Spacemacs
- ~emacs.yml~ - Base Emacs setup
- ~emacs-lsp-booster.yml~ - LSP performance enhancements

* Configuration Files

** .spacemacs

The main Spacemacs configuration file. This contains:
- Layer configurations
- Package settings
- Custom functions and keybindings
- User preferences

This file is deployed to ~~/.spacemacs~ by the Ansible setup scripts.

* Usage

After running the setup script, launch Spacemacs with:

#+BEGIN_SRC bash
emacs
#+END_SRC

On first launch, Spacemacs will install all configured packages automatically.

** DONE Alert.el Integration for Claude Code Notifications

Claude Code notifications are handled using ~alert.el~, a unified, portable
notification framework for Emacs that supports multiple backends and styles.

*** Features:
- *Persistent Log Buffer (~*log4e-alert*~)*: All Claude Code notifications are logged
  with timestamps and severity levels
- *Minibuffer Messages*: Immediate feedback appears in the Emacs minibuffer
- *Portable*: Works on any platform (Linux, macOS, Windows, Chrome OS)
- *Customizable*: Easy to switch notification styles (message, log, fringe,
  mode-line, etc.)
- *Category-based Rules*: Claude Code alerts use the "claude-code" category
  for fine-grained control

*** Usage:
#+begin_src elisp
;; View notification history
(alert--log-open-log)
;; Or switch to buffer directly
(switch-to-buffer " *log4e-alert*")

;; Customize notification style
(setq alert-default-style 'message)  ; Options: message, log, fringe, mode-line, etc.

;; Notifications appear automatically when Claude Code events occur
#+end_src

*** Implementation:
Claude Code hooks in ~spacemacs/.spacemacs~ use ~alert.el~ to send notifications
with appropriate severity levels (normal for ready/stop events, trivial for others).
All alerts are automatically logged to the ~*log4e-alert*~ buffer for persistent history.
